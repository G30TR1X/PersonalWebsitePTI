My Github

I have used github to store a couple projects, some I have finished, some are either still in development or at worst just in hiatus, either discontinued or I am planning to continue it on a later date. Check out my Github!

Here are a few of my github repo that I think worth mentioning:
  • CS50Homepage, This was from the day I was doing Harvard CS50 program back in early 2024. I manage to reach up to week 9 and this little repo was technically the first time I ever use github and the git version control. It was really the first project that introduce me to git and help me learn its antics, and from it came a fairly average looking website. It wasn't the best, but for submitting as a cs50 assignment, it was enough. If you want to see what the web looks like, you can check it our here.
  • odin-landing-page, This repo was made not long ago really. It was when I am following an open source / free course called Odin. It was a web that technically teaches how to become a fullstack developer and one of the assignment projects, well technically I see it more as a practice ground, was making this simple landing page. I am not going to lie, the template doesn't look bad really, the real test of this project was really how to use CSS, mainly CSS flexbox feature. Learning how to center a div was one of them, though this project can really does show how difficult it is to actually style a good web manually and kinda shows why frameworks exist to ease the pain a bit. If you want to see the final result of the landing page you can click this link.
